Hello, Deloitte!

This is a basic angular project to fetch hotel list from an API built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

The application allows the user to:

  • View List of Available Hotels. Default Behaviour
  • Search Hotels By Name. This is a case insensitive by name
  • Search Hotels By Keyword. This is a case insensitive filter by either name, description or location
  • Search Hotels By Minimum Rating. This is a filter for hotels rated same or higher than user selection
  • Sort Hotels. User can sort all columns
